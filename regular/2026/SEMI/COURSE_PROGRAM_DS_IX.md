## Descripción Del Curso

La asignatura consolida fundamentos previos de programación y amplía la perspectiva del estudiante hacia el diseño e implementación de soluciones bajo enfoque Full Stack. El curso trabaja aplicaciones con arquitectura cliente-servidor, servicios backend, interfaces frontend y gestión de bases de datos para que el estudiante comprenda la estructura integral de una aplicación.

Integra metodologías de programación y principios de ingeniería de software bajo estándares de calidad y seguridad informática. El objetivo es que el estudiante diseñe soluciones tecnológicas escalables, coherentes y alineadas con necesidades organizacionales reales, usando pensamiento abstracto, crítico e investigativo.

## Competencia Específica

Desarrollar soluciones de software bajo el enfoque Full Stack mediante la integración de arquitecturas cliente-servidor, servicios backend, interfaces frontend y bases de datos, aplicando metodologías de programación, principios de ingeniería de software, estándares de calidad y seguridad informática para responder a necesidades organizacionales reales.

## Resultado De Aprendizaje Principal

Integra componentes frontend, backend y bases de datos en una aplicación web funcional utilizando frameworks de desarrollo y buenas prácticas de calidad y seguridad.

## Indicadores Y Criterios De Evaluación

Diseño del proyecto:

- Formular el proyecto comprobando correspondencia entre requerimientos y objetivos.
- Determinar viabilidad técnica justificando selección de tecnologías y recursos.
- Delimitar alcance en consistencia con requerimientos.
- Diseñar estructura arquitectónica coherente e integrada.

Configuración del entorno:

- Identificar componentes y funcionalidad de una arquitectura Full Stack.
- Instalar herramientas de desarrollo según especificaciones técnicas.
- Configurar frameworks y validar integración funcional.
- Ejecutar interoperabilidad entre herramientas.

Implementación Full Stack:

- Usar módulos de Node.js para funcionalidades reutilizables en backend.
- Configurar servidores HTTP para comunicación cliente-servidor.
- Desarrollar servicios backend con Express de forma modular y escalable.

## Contenido Del Curso

### Módulo I: Formulación De Proyectos Full Stack

Duración: 2 semanas.

Unidad I: Definición del problema.

- Objetivo del proyecto.
- Alcance del proyecto.

Unidad II: Análisis de requerimientos.

- Requerimientos funcionales.
- Requerimientos no funcionales.

Unidad III: Diseño de la arquitectura Full Stack.

- Backend:
  - Diseño de API.
  - Estructura del servidor.
  - Consumo de servicios externos.
  - Pasarela de pago.
  - Consumo de modelos de IA, ML o DL.
  - Diseño de base de datos.
- Frontend:
  - Diseño de interfaz.

Unidad IV: Desarrollo y pruebas.

- Codificación técnica usando el stack.
- Verificación funcional de integración y usabilidad.

### Módulo II: Fundamentos Tecnológicos Y Entorno De Desarrollo Full Stack

Duración: 3 semanas.

Unidad I: Arquitectura del desarrollo Full Stack.

- Características del desarrollo Full Stack.
- Frameworks frontend y backend.
- Integración frontend-backend.
- Tipos de desarrollo Full Stack.
- Stacks tecnológicos más utilizados.
- Funciones del desarrollador Full Stack.
- Responsabilidades en el ciclo de desarrollo.
- Buenas prácticas.

Unidad II: Configuración del entorno y herramientas.

- Herramientas de desarrollo.
- Configuración de frameworks:
  - Backend: Node.js, npm, Express.js.
  - Frontend: React.
- Base de datos: MongoDB.

### Módulo III: Desarrollo E Implementación De Aplicaciones Full Stack

Duración: 11 semanas.

Unidad I: Desarrollo de servicios backend y seguridad de APIs.

- Node.js.
- Módulos.
- Servidores HTTP.
- Servicios backend con Express.
- Rutas GET, POST, PUT y DELETE.
- Middlewares.
- Manejo de errores y depuración.
- Desarrollo y documentación de APIs REST.
- Seguridad:
  - Cifrado de contraseñas.
  - JSON Web Token.

Unidad II: Modelado y persistencia de datos.

- MongoDB.
- Mongoose.
- Operaciones CRUD.

Unidad III: Desarrollo de interfaces y consumo de servicios.

- React.
- Estructura de aplicaciones React.
- JSX y renderizado.
- Componentes y eventos.
- Listas, formularios y validación.
- Hooks.
- Integración frontend-backend mediante consumo de APIs.
