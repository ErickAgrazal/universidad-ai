## Skill update (during this session)

New "Phase 2 — Acquire videos" priority order documented:
1. **Cloned repo audit** first (cheapest) — A3 video often sits in `presentacion_grupal/`, `Plantilla_repo_github/`, `tareas/Charla_*/` etc. even when Teams submission links elsewhere
2. **Teams SharePoint Stream** — for direct MP4 uploads via `download.aspx?UniqueId=`
3. **YouTube/external** — `yt-dlp` (private videos = ungrabable, but document credit)

New gotchas (4 added):
- Teams submission link pointing to "wrong" artifact ≠ no A3 work done — audit the cloned repo
- Git LFS pointers + stub MP4 detection (`file` + size check)
- YouTube hosting (often due to GitHub size limits) — yt-dlp + private-video fallback
- Group identification via file MD5 / PDF authorship filename pattern
